# Iniciar Tarefa: $ARGUMENTS

1. Pergunte o título e os detalhes do ticket Jira $ARGUMENTS (se eu não tiver colado).
2. Leia \`@docs/workflow/repo-bitbucket.md\` para lembrar o padrão de branch.
3. Crie a branch \`feature/$ARGUMENTS-...\` ou \`bugfix/$ARGUMENTS-...\`.
4. Leia \`@docs/tech/java-standards.md\` para carregar as regras de Java.
5. Entre em **Plan Mode** e proponha a arquitetura das classes (sem Lombok!).
